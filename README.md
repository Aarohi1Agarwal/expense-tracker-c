# Expense Tracker in C

A beginner-friendly command-line Expense Tracker built using C.

This project allows users to store and manage expenses directly from the terminal while learning core C programming concepts and GitHub workflow.

## Features

* Add Expense
* View Expense History
* Search Expense by Category
* Monthly Summary
* Persistent File Storage
* Input Validation

## Concepts Used

* Structures
* Functions
* File Handling
* Arrays and Strings
* Modular Programming
* Git and GitHub

## Project Structure

```text
expense-tracker/
├── main.c
├── expense.c
├── expense.h
├── expenses.txt
└── README.md
```

## How to Run

Compile:

gcc main.c expense.c -o tracker

Run:

./tracker
